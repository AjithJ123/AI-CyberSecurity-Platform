About PhishGuard AI

PhishGuard AI is a lightweight threat-intelligence scanner that checks links and emails against multiple industry data sources and returns a clear verdict with human-readable reasoning. Built so non-technical users can actually understand what's happening.

Detection pipeline

  • Offline URL heuristics — HTTPS, TLD, keywords, punycode, IP-host.
  • WHOIS — domain age, registrar, country, expiration.
  • Link-shortener expansion — follows bit.ly / t.co / tinyurl to the real destination.
  • Google Safe Browsing — known malicious + social-engineering URLs.
  • VirusTotal — 70+ antivirus engines + vendor category tags.
  • PhishTank — verified community phishing reports.
  • AI content review — LLM analysis of email text for social-engineering cues.
  • Email-address heuristics — typosquat detection against 30+ brands, disposable provider list, suspicious TLDs.

Limits

No tool catches everything. A "SECURE" verdict is not a guarantee. If a message asks for credentials, payment, or urgent action, verify through a channel you already trust.
