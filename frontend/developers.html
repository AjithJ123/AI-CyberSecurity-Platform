API reference

Three public endpoints. No auth in v1. Rate-limited per hashed IP.

POST /api/v1/check/url

{
  "url": "https://example.com/path"
}

POST /api/v1/check/email-address

{
  "email": "security@paypa1-verify.tk"
}

POST /api/v1/check/email

{
  "subject": "Urgent: verify your account",
  "sender":  "support@example-bank.tk",
  "body":    "Dear customer, please click..."
}
